# Inforestrepo
InfoRestrepo es una plataforma web orientada a centralizar y difundir oportunidades educativas, laborales y sociales para comunidades con acceso limitado a información institucional, con enfoque inicial en el barrio Restrepo de Bogotá. Este proyecto busca contribuir al cumplimiento del ODS 9 (Innovación, Infraestructura y Acceso a la Información).
